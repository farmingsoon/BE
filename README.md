## 📋 프로젝트 설명
### 자신의 물품의 가치를 알고 싶은 사람들을 위한 중고 거래 입찰 서비스
- 누구나 자유롭게 자신의 상품을 등록할 수 있으며 입찰에 참여할 수 있습니다.
- 사용자 편의를 위해 아래와 같은 기능들을 제공합니다.
    - 로그인(폼, 소셜)
    - 검색(키워드, 카테고리)
    - 정렬(인기, 저고가, 마감 임박 등)
    - 좋아요
    - 1 대 1 채팅
    - 알림(입찰 등록, 입찰 종료, 판매 완료, 채팅 수신)
